## المشكلة

من الـ screenshot واضح ثلاث مشاكل قاتلة بسلوك الوكيل:

1. **تسريب أخطاء تقنية للزبون**: البوت طبع حرفياً `"invalid input value" بقاعدة بيانات الطلبات...` — هذا نص خطأ Postgres خام من `submit_order` (السطر 709: `return { error: error.message }`). الـ LLM أخذ الخطأ ووضعه بالرد. كارثة احترافية.
2. **تمثيل كاذب بوجود موظف**: قال *"الموظف حالياً كدامه يثبته يدوي"* — مافي موظف فعلاً. هذا اختراع من LLM لتلطيف الفشل.
3. **Handoff عشوائي ومتكرر**: كل ما الزبون يستفسر، البوت يقول *"حولتك لزميل بشري"* بدون أي محاولة فعلية للمساعدة. القاعدة #6 بالـ prompt: *"لو الزبون متضايق/زعلان، استخدم handoff_to_human فوراً"* — تطلق بسرعة جداً، وبعد الـ handoff البوت ما يصير صامت بشكل صحيح (يبدو إن في إعادة تشغيل تلقائي أو الـ LLM يستدعي handoff في كل run جديد).

## الحل

### 1. تطهير أخطاء الأدوات (Tool error sanitization)
- بـ `agent-run/index.ts`: استبدل كل `return { error: error.message }` (في submit_order, schedule_order, set_delivery_info...) بـ:
  - تسجيل الخطأ الكامل بـ `console.error` + `agent_logs` للمالك.
  - إرجاع رسالة معيارية للـ LLM: `{ error: "TECHNICAL_FAILURE", user_message: "صار خلل بسيط بإرسال الطلب، جرّب مرة ثانية أو نحوّلك لموظف." }`
- بالـ system prompt: قاعدة صريحة **"ممنوع نهائياً ذكر أي تفاصيل تقنية (قاعدة بيانات، أخطاء، أكواد، invalid input، JSON، tokens). إذا أداة رجعت خطأ، اعتذر بسطر عام واطلب إعادة المحاولة أو اعرض الـ handoff."**

### 2. منع اختراع وجود موظف
- قاعدة جديدة بالـ prompt: **"ممنوع تقول 'الموظف يكدامه/يثبت يدوي/يراجع الآن' إلا إذا استدعيت `handoff_to_human` فعلاً بهالـ run. لا تخترع نشاط بشري وهمي."**

### 3. تشديد منطق Handoff
- عدّل القاعدة #6: handoff فقط في حالتين:
  - الزبون طلب صراحة (`موظف، انسان، حولني، بشري`).
  - فشل أداة حرجة (submit_order/preview_order) بعد إعادة محاولة واحدة.
- **مو** فقط لأنه "متضايق". أول رد على الانزعاج: اعتذار + سؤال محدد لفهم المشكلة.
- بعد `handoff_to_human` ينجح، السطر 1184 الحالي يصمت البوت. أضمن إن `is_bot_paused` ما ينعمل reset تلقائياً بأي مكان آخر (راجع dashboard + telegram-webhook).

### 4. تنظيف رسالة الـ handoff
- بدلاً من تكرار "حولتك لزميل بشري، راح يجاوبك" بكل مرة، اجعلها رسالة واحدة واضحة: *"تم تحويلك لفريق المطعم 🙏 راح يتواصل وياك أقرب وقت. ممكن تستنى هنا."*
